gu_util

A simple convenience library for POSIX C/C++ code.

Prerequisites

You need a posix system a C/C++ compiler, cmake, and a build system supported by cmake, such as Ninja or gmake.

Building

To build, you simply create a build directory (e.g. build.ninja) using cmake, then use your build system to build and install. Here is an example using Ninja: 

mkdir ../build.ninja
cd ../build.ninja
cmake -G Ninja ../gu_util
ninja
ninja install

If you require root permissions, run ninja install as root, e.g. by using sudo: 

sudo ninja install
